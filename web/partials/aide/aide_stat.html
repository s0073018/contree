Graphes

Cette écran affiche une synthèse graphique des maines enregistrées.

Deux visions sont proposées :

  • Vue type Camembert pour analyser les coups gagnants des deux équipe (contrat réussis, contrée ..) ainsi qu'un vue rapide du reste à faire pour atteindre le score gagnant (voir paramètres).

  • Vue type Histogramme pour comparer la progression des deux équipes.
